# Titanic Data Analysis Project

## 📌 Overview
This project explores the famous **Titanic dataset** using Python.  
The goal is to perform **Exploratory Data Analysis (EDA)**, uncover survival patterns, and prepare the dataset for further predictive modeling.

The dataset contains passenger details such as:
- Demographics (Age, Sex, Class)
- Family relations (SibSp, Parch)
- Ticket and Fare information
- Embarkation port
- Cabin details

---

## 🛠️ Tools & Libraries Used
- **Python 3**
- **Pandas** – data cleaning & manipulation  
- **NumPy** – numerical operations  
- **Matplotlib & Seaborn** – data visualization  
- **Jupyter Notebook** – interactive analysis environment  

---

## 🔍 Key Steps
1. **Data Loading** – Reading the Titanic dataset (`CSV` file).  
2. **Data Cleaning** – Handling missing values (Age, Cabin, Embarked).  
3. **Exploratory Data Analysis (EDA)** –  
   - Distribution of passengers by class, age, and gender  
   - Fare analysis across different classes  
   - Family size impact on survival  
   - Embarkation port statistics  
4. **Visualizations** – Graphs and charts to highlight insights.  
5. **Insights** – Patterns observed in demographics and ticket/fare data.

---

## 📊 Sample Insights
- Higher-class passengers generally paid more and had better cabin allocations.  
- Younger passengers and families had varied survival chances depending on class.  
- Missing values in **Age** and **Cabin** required imputation or exclusion strategies.
